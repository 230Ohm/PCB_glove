# Glove Data Research Collection

Goal:
Create a wearable glove data collection system for hand and finger motion research.

Known requirements:
- Five finger IMUs, one per finger.
- STM32N6-class host or controller.
- Two CAM-6GY-152VIS cameras.
- Synchronized IMU and camera data collection.
- Reliable experiment data capture.
- Clear sensor orientation and repeatable calibration.
- Wearable constraints: small size, low profile, cable strain relief, safe edges, and robust connectors.

Open questions:
- Exact STM32N6 part or module.
- Exact CAM-6GY-152VIS datasheet and connector pinout.
- Battery or wired power.
- Input voltage.
- Required IMU sampling rate.
- Required camera frame rate and resolution.
- Board location on glove.
- Connector type.
- Maximum board size.
